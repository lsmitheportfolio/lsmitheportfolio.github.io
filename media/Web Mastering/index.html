Webmastering 2014

      The information provided here encompasses the knowledge I have gained from this class, as well as some information I have learned from experience.

      Hyper Text Markup Language, paired with Cascading Style Sheets, is the language used to write computer pages. It uses tags to build objects, and it uses styles to arrange, size, and decorate them. The average person waits under half a second for a computer page to load. Large graphics can cause a delay in a web page loading. Whether an image is designed with vector tools or raster tools, it can be optimized to have a small, yet lossless file type. Adobe Photoshop is a program that includes both of these tools, so that the user can create and edit images in a variety of ways. Although images created are automatically copyrighted, they can be added to the public domain with a creative commons license.
